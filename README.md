## Project Title: AD 312 TaskManger with React State
#### By: Amde Wubshet

### Breif Description
Developed a simple task manager where users can add new tasks and mark tasks as completed. Demonstrates the correct way to update arrays and objects within the React state.

### Commands
After installing the repo, navigate into your project directory: *cd task-manager-app*. Then start the development server: *npm run dev*. Launching the webpage on a browser, you will see from top to bottom: 
- An input field, where you can enter a task by typing whatever you have in mind (e.g. chores, goals for the day, reminders, etc.). You can enter what you typed by clicking the "Update Tasks" button, and render that task in real time under the "Current Tasks" heading.
- When the task is rendered, it shows the task you inputted along with a "Complete: true/false" boolean that can be switched by pressing the "Done!" button (Initially the Complete boolean starts as false, that's why the button is named like that).
